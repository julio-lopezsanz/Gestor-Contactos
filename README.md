# Gestor de contactos en python

Proyecto en Python que implementa un gestor de contactos por consola, enfocado en practicar fundamentos clave del lenguaje como funciones, listas, cadenas de texto y estructuras de control. Este proyecto forma parte de mi proceso de aprendizaje en Python con el objetivo de:

- Reforzar el uso de funciones
- Aprender a pasar listas como parámetros
- Mejorar la lógica y el flujo de un programa
- Construir bases sólidas antes de avanzar a diccionarios y bases de datos

Características

- Menú interactivo con while
- Uso de funciones para cada acción
- Almacenamiento de contactos en una lista
- Manejo de strings (normalización de texto)
- Búsqueda y eliminación de contactos
- Validacion de cada opcion en caso de que la lista este vacia

### Funciones
- El programa permite:
- Agregar contactos (nombre y teléfono)
- Mostrar todos los contactos
- Buscar un contacto por nombre
- Eliminar un contacto
- Salir del programa
